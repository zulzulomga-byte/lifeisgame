Penting: Backup Progress Kamu!
Progress disimpan otomatis di browser ini saja (bukan di server). Kalau kamu update/upload ulang file web ini, ganti HP, ganti browser, atau cache browser ke-clear, progress yang belum di-backup bisa hilang dan gak bisa dibalikin lagi.
Cara backup & restore (ada di tab Settings):
  1. Sebelum apa pun (ganti HP, update web, dll), klik "Export Backup (JSON)" di tab Settings.
  2. Simpan file .json yang ke-download (misal di Google Drive / kirim ke diri sendiri di chat).
  3. Kalau progress hilang, klik "Import Backup" dan pilih file .json itu lagi — progress balik seperti semula.
Saran: backup tiap mau ganti konfigurasi besar atau minimal sebulan sekali.
